# 2D Segway on Moving Platform

## Description
A 2D inverted pendulum (segway) on a moving platform simulated in MuJoCo.

## Features
- 2D constrained dynamics
- Platform translation only
- Limited translational velocity
- LQR balancing controller
- ZeroMQ communication
- Moving platform disturbance

## Simulator
MuJoCo

## Controller
LQR

## Communication
ZeroMQ

## Run

pip install -r requirements.txt

python simulator.py
